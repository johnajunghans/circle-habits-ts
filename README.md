# Circle Habits

A circular habit tracker that visualizes your daily habits as arcs on a 24-hour clock wheel.

## Tech Stack

- **Vite** — build tool and dev server
- **React 19** + **TypeScript**
- **Tailwind CSS v4** — utility-first styling
- **shadcn/ui** — accessible UI components (Sheet, Dialog, Accordion, Button, Input, Label)
- **D3.js** — SVG arc generation for the habit wheel

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
